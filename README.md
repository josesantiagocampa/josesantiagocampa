## What I've been building

**🎵 Honors Thesis — Music Retrieval System Benchmark**  
Benchmarked 11 retrieval models (TF-IDF, BM25, Word2Vec, SBERT, LoRA + SimCSE) across
8 ranking metrics on a 2.75M-song Spotify dataset. Built the full evaluation framework
from scratch. Key finding: a hybrid lexical pipeline hits 61.5% HitRate@10 in under
30 minutes; adding a LoRA fine-tuned transformer gains 1% — at 4× the compute cost.  
→ [`csc498-thesis-crs`](https://github.com/josesantiagocampa/csc498-thesis-crs)

**🔭 NASA-Funded IFU Spectroscopy Analyzer** *(ongoing)*  
Production desktop app for analyzing astronomical data from JWST, VLT, and ALMA.
Parallel Gaussian fitting engine across 100K+ spatial pixels from 3GB FITS datacubes
using 8-core multiprocessing. Actively used by UA Department of Astronomy researchers.
Codebase being prepared for open-source release.

**❓ Jeopardy Open-Domain QA — Information Retrieval System**  
BM25-based IR system over 200K Wikipedia articles with regex parsing, stemming, and
tokenization. Achieved 40% Precision@1, 47.6% MRR@100, and 76% Precision@100. 
→ [`483-Jeopardy`](https://github.com/josesantiagocampa/483_Jeopardy)

**🧠 Hate Speech Detection with BERT**  
Fine-tuned BERT for hate speech classification on 4.9K labeled tweets. Identified label
ambiguity in the multiclass setup (0.44 F1), reformulated as binary classification —
pushed harmful class F1 to 0.98 at 96% accuracy despite 4:1 class imbalance.  
→ [`harmful-language-detection-sim`](https://github.com/josesantiagocampa/harmful-language-detection-sim)

**🤖 Hack Arizona — 3rd Place, Biosphere 2 Challenge**  
Multi-agent AI system that queried live environmental sensor data across Biosphere 2
biomes to generate ecosystem health reports. Built in 24 hours.
→ [`probep`](https://github.com/josesantiagocampa/probep)

**🍽️ Restaurant Point-of-Sale System**  
MVC POS system in Java with DAO pattern, JDBC, and Observer pattern for real-time GUI
sync. JUnit 5 test suite covering 6,000+ bytecode instructions across data operations,
controller logic, and multi-entity CRUD. 
→ [`CSC335_FinalProject`](https://github.com/josesantiagocampa/CSC335_FinalProject)

**🗄️ LLM Platform Database System**  
Relational schema design for a full LLM platform (users, conversations, personas,
billing, support). Full 3NF/BCNF normalization with functional dependency analysis.
Two-tier Oracle DBMS + JDBC client-server implementation.
